Winserve API Client
=======================

A api services for winserve application

Requirements
============

* PHP >= 7.*
* Laragon Application
* Sql driver for php

Installation
============

For 64bit windows machine 


Download and install --> 
[Laragon](https://sourceforge.net/projects/laragon/files/releases/4.0/laragon-full.exe) <--
